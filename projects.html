Gassed

React

Gassed is an e-commerce platform with a strategic blend of React for frontend dynamics and Node.js for a robust backend. Leveraging RESTful APIs for seamless data handling, the platform integrates Firebase for user registration, recognizing its simplicity while acknowledging real-world limitations. Targeting 18-35 year-old men, Gassed adopts a refined design with dark backgrounds, evocative imagery, and vibrant, almost neon colors for fonts and buttons.
